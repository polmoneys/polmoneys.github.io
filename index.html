What a time to be alive.

I've ditched CSS for styled components and couldn't be happier. This is my current flow, thought I'd share it with you.

TLDR;

* Props are King

* Components and styles in the same file

Pull your fav react boilerplate (create react app for example), fire your terminal and: 

  $ npm i styled-components

The most simple #win I can think of is against rainbow text designers.

Props are javascript truth so trying to bridge it with css is kind of clumsy, styled components solve glues it sensationally. 

  import styled, { css,keyframes } from 'styled-components';
      
  export default () => (
    <div>
    
      <Paragraph tint="red">
        Hello World
      </Paragraph>

      <Paragraph uppercase>
        Hello World
      </Paragraph>
    </div>
  )

  const Paragraph = styled.p`
    color: ${props => props.tint ? tint : '#444'};
    text-transform: ${props => props.uppercase ? 'uppercase' : 'none'};
  `;

The tint prop accepts any color and fallbacks to a global/default color if not present. We are cutting classes && clutter in the render().

If we transition to a more ambitious endeavour and accept the fact that the backbone of any website are links and buttons, we can go for another small effort big #win.

      
  export default () => (
    <div>
    
      <Touchable 
      onClick={e => console.log(e)}> 
        Normal Touchable 
      </Touchable>

      <TouchableBranded 
      onClick={e => console.log(e)}> 
        Branded Touchable 
      </TouchableBranded>

      <Link href="/">
        Normal Link 
      </Link>

      <LinkUnderline href="/">
        Link with Underline 
      </LinkUnderline>
    
    </div>
  )
  
  const fadeIn = keyframes`
    0% {
      opacity: 0;
    }
    100% {
      opacity: 1;
    }
  `;

  const Touchable = styled.button`
    animation: 1s ${fadeIn} ease-out;
    cursor:pointer;
    touch-action: manipulation;
    -webkit-appearance: button;
    
    & > * {
      pointer-events: none;
    }
  `;

  const TouchableBranded = Touchable.extend`
    background: pink;
    color:#fff;
 
    &:hover {
      opacity:.8;
    }
  `;

  const Link = Touchable.withComponent('a')

  const LinkUnderline = Link.extend`
    text-decoration:underline;
  `;

 <Devices>
     <Iphone

    />
     <Tablet
      pressed={this.state.device == "tablet"}
      onClick={() => this.setState({ device: "tablet" })}
